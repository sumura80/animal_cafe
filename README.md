# README
## Name: Catch Cafe

 
## Overview: 人気の動物カフェを探せるサイト
自分のコメントを投稿できるだけでなく、評価を星で登録もできる。
動物カフェの住所によりGoogleマップが表示される。
管理者によってのみカフェを登録することができる。


## Function:
使用したGems:
```
1. devise
2. bootstrap-sass
3. carrierwave(mini_magick)
4. jquery-rails
```
